# Challenge: Dali's DLL

## Description

*“First rule of the heist: no blood, no names, just strict contracts.”*

The Spanish authorities have deployed a supposedly impenetrable digital vault for the Royal Mint of Spain. Helsinki managed to extract the system's primary access binary (`client.exe`) along with the target's central banking module (`server.dll`) directly from the internal network. 

Our initial breach gets us into the front doors, allowing us to trigger the public lobby mechanisms without a hitch. But the true prize is locked behind a hidden door governed by an archaic, strictly structured Windows architectural contract.

Grab your Dalí mask and force the vault wide open.
